INTRODUCTION

Spacemacs key bindings for vim. Use <Space> as leader to provide a set of
key bindings to match the same workflow as spacemacs.

REQUIREMENTS

    hecal3/vim-leader-guide
    junegunn/fzf
    junegunn/fzf.vim
    tpope/vim-commentary
    tpope/vim-eunuch
    tpope/vim-fugitive
